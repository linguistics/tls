Proceedings

Proceedings submissions must be formatted as specified in this PDF.

We have not yet prepared the Word document template. If you would find such a template useful, please get in touch with us so that we can notify you when it is ready.

Meanwhile, we much prefer LaTeX submissions, and provide the following document class and bibliography style:

  • tls.cls
  • tlslike.bst

The following document and bibliography show how to use the document class and bib style from your LaTeX document.

  • Example.tex
  • Example.bib
  • Example-IPA.png (optional included graphic)
  • Example.pdf (the resulting document)
